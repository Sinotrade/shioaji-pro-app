## v0.1.16 — AI Agent：多模型、技能、排程任務

### AI Agent（原 AI 助理全面升級）
- **多 Provider**：Claude（Anthropic）與 GPT/Codex（OpenAI）皆可用，模型自選，金鑰存本機直連官方 API
- **預裝 shioaji 技能**：台股市場與 Shioaji 領域知識內建於 agent，天生會查行情、看持倉、懂台股規則
- **權限模式**：唯讀（只能查詢分析）／確認下單（提案卡需手動確認）／自動下單（直接送單，仍受風控引擎限制，需明確開啟）
- **技能系統**：把日常 workflow 存成具名技能，對話輸入 /技能名 直接執行；內建 /開盤巡檢、/庫存健檢、/收盤總結
- **自我學習（Hermes 式程序記憶）**：agent 完成多步驟任務後會自己把流程存成技能，下次直接調用並持續改進
- **排程與事件任務**：每天定時、每 N 分鐘、**到價觸發**、**委託成交事件觸發** — 例如「每天 08:50 開盤巡檢」「TXFR1 跌破 43000 時分析持倉風險」；執行紀錄完整保留，提案進通知中心
- 面板五頁籤：對話／技能／任務／紀錄／設定
