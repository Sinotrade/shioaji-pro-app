## v0.1.31 — Shioaji 1.7 ＋ 衍生商品探索

### Shioaji Server 1.7
- 內建 sidecar 升級至 **Shioaji 1.7.0**，全面改用 Contract V2：商品基本資料與詳細規格分流，期貨、選擇權、權證依 root 或標的快速查詢。
- 指數改用交易所編制的 master code（例如加權指數 `IX0001`），並接上新的即時指數訂閱；舊自選代碼會自動遷移。
- Contract 更新事件會主動刷新自選資料；連續月合約保留 `target_code`，server 重連後也會自動恢復行情訂閱。

### 個股期與權證
- 新增 **個股期選擇器**：輸入股票名稱或代碼，即可比較標準／小型個股期、近月／次月、所有到期月份、成交價與契約乘數。
- 新增 **權證篩選器**：依標的、認購／認售、到期區間與價內外排序，顯示現貨基準、履約價及剩餘天數。
- 兩個專用面板都能連動 K 線與下單面板，並以星號直接加入目前自選清單；另新增「衍生商品探索」版面預設。
- 一般自選搜尋不再混入大量權證；輸入「台積電期」仍可快速找到個股期。權證賣出成本試算也改用正確的 0.1% 稅率。

### 選擇權與 AI Agent
- 選擇權 T 字改用 1.7 options root API，依 `delivery_month`、`strike_price`、`option_right` 建立前月履約價表。
- AI Agent 新增商品探索工具，可理解股票名稱、個股期、權證、選擇權與正式指數代碼，並直接使用完整 Contract V2 欄位。
- 排行榜、類股熱力圖與商品搜尋改成按需載入 ContractInfo，降低啟動時的大量合約請求。
