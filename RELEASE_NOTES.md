## v0.1.15 — 閃電全開長條版型、更新後自動修復

### 修正：更新後清單空白（需手動 reload）
- 根因：自動重啟伺服器的空窗期間，一次性載入（自選清單、伺服器資訊）失敗就放棄，輪詢類面板後來自己恢復
- 現在 boot 重啟過伺服器就會在健康後自動重新載入畫面；自選清單載入改為退避重試、伺服器資訊持續重試 — 不再需要手動 reload

### ⚡ 全開：先選排版，長條優先
- 點「全開」先顯示**排版縮圖**：8 長條／6 長條／4 長條／3 長條（全高直條並排，符合閃電梯縱向形狀）＋右側直欄
- 點選後依版型自動計算座標開窗
- 規模化保護：多視窗輪詢錯峰抖動（避開帳務 API 速率限制）、成交事件刷新抖動（八個視窗不再同瞬間齊發）、控制列在 240px 窄條下自動換行

### 動態鋪單防重複
- 修正補單競態：剛送出的單還沒出現在委託回報前，不會在同價位重複補單（15 秒護欄）

### 介面
- 全 App icon 改用 Lucide（開源、專業線條風格），不再使用 emoji：面板標題列（連動/鎖定/彈出/關閉）、自選清單、托盤、主題選單、通知中心等
- 主題深淺色同步原生視窗外觀（macOS／Windows 標題列）
